# css-nest-comment

## 简介/description
和前面的`html-nest-comment`一样，这个插件也是用来方便注释`css`的
a plugin that can help you bypass exist css comment, works like the last one plugin: `html-nest-comment`

## 使用/how to use
**在你使用之前，你需要选择你想要注释的区域 / Before use, you need to select a block which you want**
1. 鼠标右键弹出目录，在目录中选择`CSS Nest Comment`（推荐）。Click right mouse button and select the selection `CSS Nest Comment`(recommend)
2. 使用快捷键，在下面
3. 通过`F1`选择`CSS Nest Comment Out`. / keydown F1 and select `CSS Nest Comment Out`.

## shortcut key
- `windows`: `ctrl + shift + '`
- `Mac`: `cmd + shift + '`
